music_bot создан для воспроизведения звука из видео загруженных на платформе Youtube

!Для запуска программы необходимо установить и поместить в папку файлы ffmpeg.exe, ffplay.exe, ffprobe.exe с официального сайта https://ffbinaries.com/downloads!

Бот имеет различные команды:

+join - присоеденение бота к голосовому чату, где находится автор обращения
+disconnect - выход из голосового чата
+add {url} - добавление видео в очередь
+play {url} - начать воспроизведеие видео
+loop - видео играет повторно
+unloop - прекратить повторное воспроизведение
+queue - запросить очередь видео
+pause - приостанавливает видео
+skip - начинает проигрыш следуещего в списке видео
+clear - очистить список
